# Live Free Initiative - Professional Website

## Overview
A modern, professional, and fully responsive website for **Live Free Initiative**, a youth-founded Christian community-based organization in Adjumani District, Uganda focused on empowering young people and promoting drug-free living.

## 📁 Project Structure

```
LFI/
├── index.html              # Home page
├── about.html              # About Us page
├── programs.html           # Programs & Services page
├── projects.html           # Impact & Projects page
├── blog.html               # News & Blog page
├── contact.html            # Contact & Support page
├── css/
│   └── styles.css          # Main stylesheet (comprehensive)
├── js/
│   └── script.js           # JavaScript functionality
├── images/                 # Image assets (to be added)
└── README.md              # This file
```

## 🔧 Customization Guide

### Changing Colors
Edit the CSS variables at the top of `css/styles.css`:
```css
:root {
    --primary-color: #0066cc;
    --secondary-color: #22c55e;
    --accent-color: #f59e0b;
    /* ... other variables */
}
```

### Adding Images
1. Place images in the `images/` folder
2. Replace image placeholders in HTML with actual image paths
3. Update `alt` attributes for accessibility

### Modifying Content
All HTML files can be edited directly. Key sections to update:
- Organization name, location, contact info
- Mission and vision statements
- Programs and services descriptions
- Team member information
- Blog posts and news

### Adding New Pages
1. Create new HTML file based on existing page structure
2. Link in navigation menu
3. Style with existing CSS classes
