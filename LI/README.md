# Light International Website

This is a static website for Light International, a child-centered non-profit organization in Uganda.

## Overview
The website includes:
- Home page with overview
- About Us page with vision, mission, values, etc.
- Programs page detailing education, health, and protection programs
- Contact page with information and form

## Features
- Responsive design
- Modern UI with gradients, shadows, hover effects, and animations
- Contact form with basic validation
- Image gallery with relevant photos
- Smooth scrolling and fade-in effects
- Professional typography

## How to Run
1. Open `index.html` in a web browser.
2. Navigate using the menu.

## Files
- `index.html`: Home page
- `about.html`: About Us
- `programs.html`: Programs
- `contact.html`: Contact
- `style.css`: Stylesheet
- `script.js`: JavaScript
- `images/`: Folder for images (add hero.jpg for background, logo.png, etc.)

## Customization
- Add images to `images/` folder and reference in HTML/CSS.
- Modify content in HTML files.
- Enhance CSS for better design.
- For production, consider using a framework like Bootstrap or hosting on a server.

## Troubleshooting
- Ensure all files are in the same directory.
- If links don't work, check file paths.
- For form submission, currently shows an alert; integrate with backend for real sending.
